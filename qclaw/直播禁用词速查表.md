# 直播禁用词速查表 🚫

## 一、绝对化用语（最常见违规）

| ❌ 禁用词 | ✅ 替代说法 |
|-----------|-----------|
| 最、最佳、最优 | 很好、非常棒、很不错 |
| 第一、首个、唯一 | 领先、前列、出色 |
| 顶级、顶尖、极致 | 优秀、高品质、出色 |
| 最便宜、最低价 | 价格实惠、性价比高 |
| 100%、绝对、肯定 | 大概率、很多朋友反馈 |
| 史无前例、前所未有 | 非常少见、难得 |
| 全球领先、全国第一 | 行业领先、口碑很好 |
| 极品、绝版、王牌 | 热门款、畅销款 |
| 销冠、爆款之王 | 热销款、受欢迎 |
| 万能、万能的 | 多功能、实用 |

## 十、违规后的处理

| 违规等级 | 后果 | 处理方式 |
|---------|------|---------|
| 轻微 | 限流降权 | 停播1-2天，检查内容，删除违规片段 |
| 中度 | 短期禁播 | 停播3-7天，学习平台规则，申诉 |
| 重度 | 长期封号 | 提交申诉，等待审核，必要时重新注册 |

> 💡 **建议**：每次直播录屏，违规时有证据可以申诉。
