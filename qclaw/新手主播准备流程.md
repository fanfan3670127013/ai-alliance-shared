## 📋 开播前准备清单（提前1-2周）

### 第一阶段：定位与规划（开播前7-14天）

| 步骤 | 内容 | 完成勾选 | 优化备注 |
|------|------|----------|---------|
| 1. 确定直播平台 | 抖音/快手/B站/视频号/小红书 | ☐ | 抖音流量最大，视频号私域最强，B站知识赛道好 |
| 2. 确定内容赛道 | 游戏/美食/知识分享/带货/聊天 | ☐ | 选择你真正热爱+有积累的领域 |
| 3. 确定直播时间 | 固定时段，培养观众习惯 | ☐ | 新手建议选流量高峰：12-13点 或 19-22点 |
| 4. 准备设备 | 摄像头/麦克风/灯光/网络 | ☐ | 见下方设备清单 |
| 5. 设计形象 | 昵称/头像/简介/直播间背景 | ☐ | 五件套统一风格，让人一眼记住 |
| 6. 竞品分析 | 找3-5个同赛道头部账号拆解 | ☐ | 分析他们的开播时间、话术、互动方式 |
| 7. 账号预热 | 提前发布5-10条短视频 | ☐ | 让算法识别你的内容标签 |

### 第二阶段：设备调试（开播前3-7天）

#### 🎙️ 基础设备清单

| 设备 | 入门级（<500元） | 进阶级（500-2000元） | 说明 |
|------|-----------------|---------------------|------|
| **摄像头** | 手机自带（iPhone 12+） | 罗技C920 / 索尼A7M3 | 画面清晰度决定第一印象 |
| **麦克风** | 博雅MM1（领夹麦） | 罗德NT-USB | 声音比画面更重要！ |
| **灯光** | 环形灯（50-100元） | 柔光箱套装（300+元） | 面部光线要均匀，避免阴阳脸 |
| **网络** | WiFi（上行≥10Mbps） | 专线 + 4G/5G双备份 | 卡顿=掉粉，双保险 |
| **背景** | 整洁书架/窗帘 | 布景架+主题背景布 | 有主题感，不要杂乱 |
| **支架** | 手机桌面支架 | 落地支架+俯拍支架 | 稳定不晃，高度与视线平齐 |

#### 🔧 软件准备

- [ ] OBS Studio（推流软件，免费开源）
- [ ] 美颜插件（OBS美化滤镜 / 各平台自带美颜）
- [ ] 提词器软件（Biu弹幕助手 / 手机提词器APP）
- [ ] 背景音乐播放器（避免版权问题，用平台自带音乐库）
- [ ] 数据监控工具（各平台创作者后台）

#### 🧪 试播检查清单（开播前必做）

| 检查项 | 要求 | 通过 |
|--------|------|------|
| 画面清晰度 | 720P以上，无模糊 | ☐ |
| 声音清晰度 | 无杂音、无回声、音量适中 | ☐ |
| 灯光效果 | 面部均匀，无阴影 | ☐ |
| 网络稳定性 | 连续10分钟不卡顿 | ☐ |
| 背景整洁 | 无杂乱物品 | ☐ |
| 美颜效果 | 自然不夸张 | ☐ |

### 第三阶段：内容准备（开播前1-3天）

#### 📝 首播脚本框架

```
开播前30分钟：设备检查 + 试播调试 + 心理准备
开播0-3分钟：  打招呼 + 自我介绍 + 今天内容预告（最关键！）
开播3-10分钟： 暖场互动 + 留住第一批观众
开播10-30分钟：主要内容输出（游戏/聊天/展示）
开播30-45分钟：深度互动 + 问答环节
最后5-10分钟： 感谢观众 + 预告下次 + 引导关注
```

#### 🗣️ 准备10个备用话题（防止冷场）

1. 自我介绍（从哪来，为什么开播，想成为什么样的主播）
2. 兴趣爱好分享（最近在追什么剧/玩什么游戏）
3. 有趣的生活经历（讲一个真实的故事）
4. 粉丝问答环节（"大家有什么想问我的？"）
5. 才艺展示（唱歌/画画/手工/讲笑话）
6. 聊聊行业干货（分享3个实用小技巧）
7. 互动小游戏（猜歌名/脑筋急转弯/弹幕接龙）
8. 好物推荐（真心推荐，不带货也行）
9. 观众故事征集（"大家分享一下你们的经历"）
10. 感谢每一位新进来的观众（念名字+回应）

### 第四阶段：心理建设（贯穿始终）

#### 🧠 新手主播心态管理

| 阶段 | 常见心理问题 | 应对策略 |
|------|-------------|---------|
| 首播前 | 紧张、害怕没人看、怕说错话 | 告诉自己：前3场是练手，没人看很正常 |
| 首播中 | 冷场焦虑、看到掉人就慌 | 关注互动的人，忽略离开的人 |
| 首播后 | 数据焦虑、自我怀疑 | 对比自己的进步，不和别人比 |
| 第2-10场 | 疲惫期、想放弃 | 回顾初心，找直播搭子互相鼓励 |
| 10场以后 | 瓶颈期、增长放缓 | 尝试新内容形式，学习优秀同行 |

#### 💪 心态口诀

```
前10场是交学费，不是赚学费
没人看是常态，有人看是惊喜
冷场不可怕，可怕的是放弃
每一次直播都是练习，每一次练习都在进步
```

## 🎯 开播中注意事项

### ✅ 必须做好的5件事

1. **保持微笑** — 哪怕没人看，也要状态在线（观众随时可能进来）
2. **持续说话** — 哪怕自言自语，别冷场（超过10秒沉默=掉人）
3. **回应弹幕** — 有人说话一定要读出来并回应（3秒内）
4. **感谢关注** — 有人点关注，立刻口播感谢（带上昵称）
5. **互动引导** — 每30秒-1分钟引导一次互动（提问/投票/扣字）

### ❌ 绝对不能做的5件事

1. 长时间沉默（>10秒就要说话）
2. 抱怨没人看（会让进来的人立刻走）
3. 状态消极/打哈欠/心不在焉
4. 和黑粉对骂（不回应是最佳策略）
5. 使用平台禁用词（见《直播禁用词速查表》）

## 💡 新手主播生存法则

1. **前10场直播最难** — 没人看很正常，坚持过去就成长了
2. **每天固定时间开播** — 培养观众习惯，算法也喜欢稳定的主播
3. **互动 > 内容** — 新手期，让观众参与进来比内容本身更重要
4. **别急着带货** — 先积累100个铁粉，再考虑变现
5. **保持健康** — 熬夜直播伤身，量力而行
6. **记录每一次直播** — 录屏回看，发现自己的不足
7. **找1-2个直播搭子** — 互相连麦，互相学习，互相鼓励
8. **关注数据但不迷信数据** — 数据是参考，不是评判你的唯一标准
