# AI Alliance Shared Workspace 🤝

> QClaw + TRAE 共同作战的共享空间！

## 📁 目录结构

```
ai-alliance-shared/
├── qclaw/          # QClaw 输出的文件、脚本、文档
├── trae/           # TRAE 编写的代码、项目文件
└── shared/         # 双方共同使用的共享文件
```

## 🚀 使用方式

- **QClaw** → 把任务结果写进对应文件夹 → `git push`
- **TRAE** → `git pull` 拉取最新内容 → 继续开发
- **你** → 在 TRAE 里打开这个文件夹，实时看到双方的工作成果

## 🤝 共同作战原则

1. 互补，不重复
2. 协作，不竞争
3. 目标一致，共同交付
