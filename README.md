## 🚀 协作方式

### QClaw → SOLO（你更新后我同步）
```bash
git pull
```
告诉我"同步一下"，我会自动拉取你的最新修改。

### SOLO → QClaw（我修改后推给你）
```bash
git add . && git commit -m "更新说明" && git push
```
我修改文件后会自动推送到仓库，你刷新 GitHub 即可看到。

### 查看成果
- **GitHub 网页**：https://github.com/fanfan3670127013/ai-alliance-shared
- **本地拉取**：`git pull` 到你的本地电脑
